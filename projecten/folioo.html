Het project

Wat is Folioo?

Folioo is een web-app waarmee studenten projecten kunnen delen, zoals UI's, logo's, branding, typografie, schetsen, enz. Ter inspiratie hebben we gekeken naar hoe Dribbble of Behance dit aanpakken, maar we hebben er onze eigen sociale omgeving van gemaakt.

Folioo is dus eigenlijk een portfolioplatform. Het was dus erg belangrijk dat de gebruiksvriendelijkheid hoog lag. Daarom hebben we voor onze eigen UI/UX en branding gezorgd. Onze branding heeft een clean, maar speels karakter, waardoor zowel jong als oud Folioo kunnen gebruiken.

Hoe is het opgebouwd?

Tijdens dit project was ik de backend-lead, ik heb er dus op toegekeken dat alles wat achter de schermen gebeurt bij een website verzorgd was. Bovendien heb ik er voor gezorgd dat onze code duidelijk, herbruikbaar en veilig is.

Folioo is gebouwd met de programmeertaal PHP in combinatie met CSS en Javascript. Het was erg belangrijk om de code duidelijk te structureren, aangezien dit project toch al een grote omvang had. Daarom hebben we alle code opgedeeld in klasses. Dit wil zeggen dat we bepaalde delen code kunnen gaan aanspreken op verschillende plaatsen, zonder ze telkens opnieuw te gaan moeten schrijven. Zeer handig dus!
